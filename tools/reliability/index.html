Reliability engineering

Quantify failure exposure before you generalize it.

Reliability metrics only compare cleanly when the population, operating exposure, failure definition, and time basis are held in view. Start with observed data, then move deliberately into probability or system architecture.

Use probability models with care.

The mission and system tools assume that the input probabilities refer to the same mission duration and that component failures are independent. Field data often violates one or both assumptions through common-cause failure, shared utilities, changing duty, or incomplete detection.

Use the calculators to expose the math, then document the engineering judgment that surrounds it. The reference desk summarizes the core formulas.
