Use the same scope on each option.

Modeling is most valuable when it makes omissions visible. Check whether each option contains production impact, safety/environmental exposure, energy, labor, spares, installation downtime, and residual risk. Avoid mixing revenue with contribution margin.

These first-pass tools do not discount cash flows or model uncertainty. A major capital decision should progress to a reviewed lifecycle-cost model with financial assumptions agreed by the business.
